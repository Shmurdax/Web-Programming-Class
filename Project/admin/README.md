# Admin Dashboard

Modern admin dashboard for managing products, inventory, and sales.

## Access

Open in browser: `admin/index.html` or `http://localhost:YOURPORT/admin/index.html` (if serving from backend)

## Features

- **Dashboard**: Overview with stats and recent sales
- **Products**: Create, edit, delete products
- **Inventory**: Track stock levels with minimum stock alerts
- **Sales**: Record sales and automatically update inventory

## Usage

1. Navigate using the sidebar menu
2. Click "Add" buttons to create new entries
3. Edit or delete existing entries
4. Data updates automatically every 5 seconds

## API Integration

All data syncs with the backend API at `http://localhost:5000/api/`
