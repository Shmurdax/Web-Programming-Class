## 🛠️ Technologies Used

- **HTML5** - Semantic markup and page structure
- **CSS3** - Custom styling, Flexbox, Grid, gradients, animations, and responsive design
- **Google Fonts** - Merriweather and Alfa Slab One
- **Vanilla JavaScript** - Animated years counter on About page

No frameworks or external libraries were used.

## 📱 Responsive Design

The website is fully responsive and optimized for:
- Mobile phones (< 576px)
- Tablets (576px – 991px)
- Desktop screens

## 📄 Pages Included

1. **Home** (`index.html`) – Hero banner, services overview, testimonial
2. **About Us** (`about.html`) – Family story, dad photo, years counter
3. **Services and Prices** (`prices.html`) – Product listings with real prices
4. **Contact** (`contact.html`) – Contact information and links

## 🚀 How to Run Locally

1. Clone or download the repository
2. Open the project folder
3. Double-click `index.html` or open it in your browser
4. (Recommended) Use **Live Server** in VS Code for best experience
